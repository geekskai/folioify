# Hello World!

Welcome to our brand new blog. This is our very first post created using the MDX system.

## What is MDX?

MDX is an extension of Markdown that allows you to include JSX components directly in your Markdown content. This makes writing interactive and dynamic content much easier.

Here's an example of some code:

```javascript
function greeting(name) {
  return `Hello, ${name}!`;
}

console.log(greeting('Reader'));
```

## Features of Our Blog

Our blog supports:

- **Rich Markdown formatting** with headings, lists, code blocks
- **Syntax highlighting** for code examples
- **Tag categorization** for organizing content
- **Reading time estimates** to help readers plan their time
- **Responsive design** that works on all devices
